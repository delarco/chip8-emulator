# CHIP-8 Emulator

A [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) emulator written in TypeScript.

You can test it here https://delarco.com.br/games/chip8-emulator.


## Roadmap

  - [X] CHIP-8 CPU

  - [ ] User Interface

- [ ] Debugger
- [ ] Input
  - [X] Support custom mappings
  - [X] Support gamepads
